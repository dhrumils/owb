PDM

  • The PDM page is selected by clicking the PDM tab next to the gateway tab. The PDM tab is highlighted as shown in figure.


  • There is an UPLOAD button on the right side of the page.


  • By clicking the UPLOAD button the files from the pc will be browsed.
  • Select the files to be uploaded and the table will be updated.

  • The table shows the file name, date when the file is updated, remarks, the version of the file.

  • Also there are actions like edit and delete available for each file. The file name can be edited by clicking the pencil icon in the actions column. The file will be deleted by clicking the trash icon in the actions column.

  • The file which is to be updated in the system is to be selected by clicking the appropriate radio button placed before the file name.


  • Once the radio button is selected the Save & Cancel button at the bottom will be active. If the file is to be uploaded in the system then click the Save button otherwise click Cancel button.
